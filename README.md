# nodejs_complete_preparation
This is a complete preparation resource for backend web developers who are willing to excel themselves in node.js

## day 01 --> hello node.js

- covered fs module --> append, copy, read, write etc.
- what is package.json --> how to set the configuration. Start command!
- require & exports: use require('./module') to import CommonJS modules; expose values with exports.name = value or module.exports = value (anonymous export via module.exports = function(...) { ... }).
- making modules & using functions: put functions in a separate .js file, export them (named or anonymous), then require the module and call the exported functions in your app.
- handling MODULE_NOT_FOUND error in the terminal.
